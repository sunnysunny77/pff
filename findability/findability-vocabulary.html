Findability Vocabulary

Glossary


  • Crawling
    • How search engines discover a web page
  • Indexing
    • Storage of organised web content found during crawling
  • KPI
    • Key performance indicators show a measure towards achieving a goal
  • Organic Traffic
    • Page traffic from search results
  • Metric
    • Metics provide information about a process or activity from numbers
  • Ranking
    • Order of search results from a query
  • Back links
    • Links to a web page from other 3rd party domains
  • Content
    • The information on a page, can be multimedia that is there for the audience

  • Findability

    • Optimising a web site structure and content for organic search rankings
  • Keyword
    • The targeted word or phrase that relate to content that are likely to be used as queries
